# DormGuard — 宿舍电费监控

> 专为西华大学设计的宿舍电费守望者。余额低了自动喊你，避免深夜突然断电。

## 它能干啥

- 定时自动查宿舍还剩多少电费
- 余额掉到警戒线时，QQ 机器人立刻私聊提醒你
- 告警群同步推送，室友也能收到通知
- 历史电费记录随时回看

## 怎么实现的

| 那层 | 用的啥 |
|------|--------|
| 后端 | Python + FastAPI |
| 前端 | Vue 3 |
| 数据库 | MySQL |
| 消息推送 | QQ 机器人（go-cqhttp） |
