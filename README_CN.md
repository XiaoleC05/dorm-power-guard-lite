# DormGuard — 宿舍电费监控

> 西华大学宿舍电费监控工具，余额低于阈值时自动告警。

## 简介

本项目为西华大学学生设计，实时监控宿舍电费余额，避免因欠费导致停电。

## 功能

- 定时自动查询宿舍电费余额
- 余额低于设定阈值时通过 QQ 机器人发送告警
- 告警群通知，室友同步接收提醒
- 历史电费数据记录与查询

## 技术栈

| 层级 | 技术 |
|------|------|
| 后端 | Python + FastAPI |
| 前端 | Vue 3 |
| 数据库 | MySQL |
| 消息推送 | QQ Bot（go-cqhttp） |
