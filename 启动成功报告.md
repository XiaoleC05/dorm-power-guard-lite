## 📝 下一步操作

### 1. 配置爬虫Token（必须）

系统需要配置爬虫Token才能自动抓取电费数据：

1. 参考 `backend/CRAWLER_GUIDE.md` 使用抓包工具获取Token
2. 编辑 `backend/.env` 文件
3. 添加或修改：
   ```env
   CRAWLER_TOKEN=你的token
   CRAWLER_DORM_NUMBER=你的宿舍号
   ```
4. 重启后端服务

### 2. 测试爬虫功能

```powershell
cd backend
python test_crawler.py
```

### 3. 配置告警规则

访问前端界面 http://localhost:3000，在"告警规则"页面添加规则。

### 4. 配置告警通知（可选）

- **邮件告警**: 在 `.env` 中配置SMTP信息
- **QQ机器人**: 配置go-cqhttp并在 `.env` 中配置
