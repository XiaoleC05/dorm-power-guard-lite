# 快速开始指南

## 🚀 三步快速启动

### 第一步：安装依赖

**Windows:**
```cmd
cd backend
install.bat
```

**Linux/Mac:**
```bash
cd backend
chmod +x install.sh
./install.sh
```

### 第二步：配置数据库

1. 创建MySQL数据库：
```sql
CREATE DATABASE dorm_power_guard CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

2. 配置 `.env` 文件：
```cmd
cd backend
copy .env.example .env  # Windows
# 或
cp .env.example .env    # Linux/Mac
```

编辑 `.env`，至少配置：
```env
DB_HOST=localhost
DB_PORT=3306
DB_USER=root
DB_PASSWORD=your_password
DB_NAME=dorm_power_guard
```

### 第三步：启动服务

**Windows:**
```cmd
cd backend
start.bat
```

**Linux/Mac:**
```bash
cd backend
chmod +x start.sh
./start.sh
```

或直接运行：
```bash
python run.py  # 或 python3 run.py
```

## ✅ 验证安装

1. **访问API文档**: http://localhost:8000/docs
2. **健康检查**: http://localhost:8000/health
3. **测试爬虫**: `python test_crawler.py`

## 📝 下一步配置

1. **配置爬虫**（必须）：
   - 参考 `backend/CRAWLER_GUIDE.md` 抓包获取Token
   - 配置 `CRAWLER_TOKEN` 和 `CRAWLER_DORM_NUMBER`

2. **配置告警**（可选）：
   - 邮件告警：配置SMTP信息
   - QQ机器人：配置go-cqhttp
