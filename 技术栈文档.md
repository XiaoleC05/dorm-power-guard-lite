```python
def crawl_and_save(db: Session, force_alert: bool = False) -> bool:
    enabled_rules = db.query(AlertRule).filter(AlertRule.enabled == True).all()
    for rule in enabled_rules:
        if not rule.room_id:
            continue
        data = crawler.fetch_power_data(
            dorm_number=rule.dorm_number, 
            room_id=rule.room_id
        )
        # 保存数据并检查告警
```

## 告警系统

### 告警类型

1. **邮件告警**
   - 发送方：全局配置 `EMAIL_FROM`
   - 接收方：告警规则中的 `email_address`
   - 必须配置：启用邮件告警时必须填写接收邮箱

2. **QQ告警**
   - 发送方：全局配置 `QQ_BOT_SENDER_ID`（NapCatQQ登录的QQ号）
   - 接收方：告警规则中的 `qq_receiver_id`
   - 必须配置：启用QQ告警时必须填写接收QQ号或群号
   - 自动判断：群号（>=1000000000）或用户QQ号

### 防频繁告警机制

- **时间窗口**: 1小时内不重复告警
- **独立控制**: 每个告警类型（email/qq）和类别（ac/light）独立控制
- **失败重试**: 如果上次告警失败，允许立即重试
- **强制告警**: 手动触发时忽略防频繁限制

### 告警消息格式

```

当前余量：11.14 度
告警阈值：20.0 度

【完整余量信息】
  空调余量：15.00 度
  照明余量：11.14 度 ⚠️
```

## 多宿舍支持

### 实现原理

1. **告警规则配置**
   - 每个宿舍创建独立的告警规则
   - 规则中包含 `room_id`（房间ID）和 `dorm_number`（宿舍号）

2. **爬虫遍历**
   - 爬虫服务获取所有启用的告警规则
   - 为每个规则使用对应的 `room_id` 获取电费数据

3. **数据存储**
   - 每个宿舍的数据独立存储
   - 通过 `dorm_number` 字段区分

### 使用流程

1. **创建告警规则**
   - 填写宿舍号（如：324）
   - 填写房间ID（通过抓包获取）
   - 配置告警阈值和接收方

2. **自动获取数据**
   - 定时任务触发
   - 遍历所有启用的规则
   - 获取每个宿舍的电费数据

3. **监控面板显示**
   - 显示所有已启用规则的宿舍
   - 每个宿舍独立显示卡片

### 调度配置

- **调度器**: APScheduler
- **执行时间**: 每天 8:00, 12:00, 18:00, 22:00（可配置）
- **任务**: 爬取电费数据并检查告警

### 任务流程

1. 获取所有启用的告警规则
2. 遍历每个规则，获取电费数据
3. 保存数据到数据库
4. 检查是否需要告警
5. 发送告警（如果满足条件）

## 部署工具

### Nginx

**用途**：Web服务器和反向代理

**配置示例**：
```nginx
server {
    listen 80;
    server_name your_domain.com;

    location / {
        root /opt/dorm-power-guard-lite/frontend/dist;
        try_files $uri $uri/ /index.html;
    }

    location /api {
        proxy_pass http://127.0.0.1:8000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
```

### systemd

**用途**：Linux系统服务管理

**服务文件示例**：
```ini
[Unit]
Description=Dorm Power Guard Backend Service
After=network.target mysql.service

[Service]
Type=simple
User=www-data
WorkingDirectory=/opt/dorm-power-guard-lite/backend
ExecStart=/usr/bin/python3 /opt/dorm-power-guard-lite/backend/run.py
Restart=always

[Install]
WantedBy=multi-user.target
```
