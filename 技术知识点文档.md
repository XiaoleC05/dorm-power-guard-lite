### 2. FastAPI 0.104+

**用途**：现代、快速的Web框架，用于构建REST API

**特点**：
- 基于Python 3.6+的类型提示
- 自动生成API文档（Swagger UI）
- 高性能，基于Starlette和Pydantic
- 支持异步请求处理

**安装**：

```bash
pip install fastapi
```

**基本使用**：

```python
from fastapi import FastAPI

app = FastAPI()

@app.get("/")
def read_root():
    return {"message": "Hello World"}

@app.get("/items/{item_id}")
def read_item(item_id: int):
    return {"item_id": item_id}
```

**项目中的使用**：
- 定义API路由（`app/api/`目录）
- 处理HTTP请求和响应
- 自动数据验证（结合Pydantic）
- 生成API文档：访问 `http://localhost:8000/docs`

**关键特性**：
- **依赖注入**：使用`Depends()`注入数据库会话
- **路径参数**：`@app.get("/items/{item_id}")`
- **查询参数**：`@app.get("/items?skip=0&limit=10")`
- **请求体**：使用Pydantic模型验证
- **响应模型**：使用Pydantic模型序列化

**API文档访问**：
- Swagger UI：`http://localhost:8000/docs`
- ReDoc：`http://localhost:8000/redoc`

**最佳实践**：
- 使用Pydantic模型进行数据验证
- 使用依赖注入管理数据库连接
- 使用异步函数提高性能
- 合理使用路由分组（APIRouter）

### 4. SQLAlchemy 2.0+

**用途**：Python ORM（对象关系映射）框架，用于数据库操作

**特点**：
- 支持多种数据库（MySQL、PostgreSQL、SQLite等）
- 强大的查询构建器
- 支持数据库迁移
- 类型安全

**安装**：

```bash
pip install sqlalchemy
```

**基本使用**：

```python
from sqlalchemy import create_engine, Column, Integer, String
from sqlalchemy.ext.declarative import declarative_base
from sqlalchemy.orm import sessionmaker

Base = declarative_base()

class User(Base):
    __tablename__ = 'users'
    id = Column(Integer, primary_key=True)
    name = Column(String(50))

# 创建引擎
engine = create_engine('mysql+pymysql://user:pass@localhost/db')

# 创建会话
SessionLocal = sessionmaker(bind=engine)

# 使用会话
db = SessionLocal()
user = User(name="John")
db.add(user)
db.commit()
```

**项目中的使用**：
- 定义数据库模型（`app/models.py`）
- 数据库连接管理（`app/database.py`）
- CRUD操作（通过Session）

**核心概念**：
- **Engine**：数据库连接引擎
- **Session**：数据库会话，用于执行操作
- **Model**：数据库表的Python类表示
- **Query**：查询构建器

**最佳实践**：
- 使用会话依赖注入管理数据库连接
- 使用事务确保数据一致性
- 合理使用索引提高查询性能
- 使用连接池管理数据库连接

### 6. APScheduler 3.10+

**用途**：Python定时任务调度库

**特点**：
- 支持多种触发器（Cron、Interval、Date）
- 支持后台任务执行
- 支持任务持久化（可选）

**安装**：

```bash
pip install apscheduler
```

**基本使用**：

```python
from apscheduler.schedulers.background import BackgroundScheduler
from apscheduler.triggers.cron import CronTrigger

scheduler = BackgroundScheduler()

# Cron表达式：每天8点执行
scheduler.add_job(
    func=my_task,
    trigger=CronTrigger(hour=8),
    id='daily_task'
)

scheduler.start()
```

**项目中的使用**：
- 定时执行爬虫任务（`app/scheduler.py`）
- 默认时间点：8:00, 12:00, 18:00, 22:00

**Cron表达式示例**：
- `hour=8`：每天8点
- `hour=8,12,18,22`：每天8点、12点、18点、22点
- `hour=8, minute=30`：每天8点30分
- `day_of_week='mon-fri', hour=9`：工作日9点

**最佳实践**：
- 使用BackgroundScheduler在后台运行
- 合理设置任务执行时间，避免服务器压力
- 处理任务异常，避免影响调度器

```python
from pydantic import BaseModel, Field, EmailStr

class User(BaseModel):
    id: int
    name: str = Field(..., min_length=1, max_length=50)
    email: EmailStr
    age: int = Field(..., ge=0, le=120)

# 验证数据
user = User(id=1, name="John", email="john@example.com", age=25)

# 转换为字典
user_dict = user.model_dump()

# 转换为JSON
user_json = user.model_dump_json()
```

**项目中的使用**：
- API请求/响应模型（`app/schemas.py`）
- 配置管理（`app/config.py`，使用pydantic-settings）

**核心特性**：
- **类型验证**：自动验证数据类型

- **序列化**：自动转换为字典/JSON
- **文档生成**：自动生成API文档

**最佳实践**：
- 使用类型提示定义模型
