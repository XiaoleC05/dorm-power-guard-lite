# 🚀 抓包快速入门（5分钟版）

## 什么是抓包？

**简单说**：抓包就是"偷看"手机小程序和服务器之间的对话，了解小程序是如何获取数据的。

## 为什么需要抓包？

小程序不告诉我们API地址，我们需要通过抓包来：
- 找到API地址（小程序向哪个网址要数据）
- 获取Token（认证凭证，相当于"密码"）
- 了解数据格式（请求和响应的格式）

## 🎯 最简单的方法：使用Charles（推荐）

### 步骤1：下载安装（2分钟）

1. 访问：https://www.charlesproxy.com/download/
2. 下载Windows版本
3. 安装（一路下一步）
4. 启动，选择"14天免费试用"

### 步骤2：配置Charles（2分钟）

1. **启用HTTPS抓包**
   - 菜单：`Proxy` → `SSL Proxying Settings...`
   - ✅ 勾选 `Enable SSL Proxying`
   - 点击 `Add`，Host填 `*`，Port填 `443`
   - 点击 `OK`

2. **查看端口**
   - 菜单：`Proxy` → `Proxy Settings...`
   - 记住端口号（默认 `8888`）

### 步骤3：手机连接（3分钟）

1. **查看电脑IP**
   - 按 `Win+R`，输入 `cmd`，回车
   - 输入 `ipconfig`，回车
   - 找到 `IPv4 地址`，例如：`192.168.1.100`

2. **手机设置代理**
   - **Android：**
     - 设置 → WiFi → 长按当前WiFi → 修改网络
     - 高级选项 → 代理 → 手动
     - 主机名：`192.168.1.100`（你的电脑IP）
     - 端口：`8888`
     - 保存
   
   - **iPhone：**
     - 设置 → WiFi → 点击WiFi右侧的 `ⓘ`
     - HTTP代理 → 配置代理 → 手动
     - 服务器：`192.168.1.100`（你的电脑IP）
     - 端口：`8888`
     - 存储

3. **安装证书**
   - 手机浏览器访问：`chls.pro/ssl`
   - 下载并安装证书
   - **iPhone重要**：设置 → 通用 → 关于本机 → 证书信任设置 → 开启信任

### 步骤4：开始抓包（1分钟）

1. 打开微信小程序（电费查询）
2. 执行查询操作
3. 在Charles左侧查看请求列表
4. 找到包含"power"、"电费"、"balance"的请求
5. 点击请求，查看右侧详细信息

### 步骤5：提取信息（1分钟）

点击请求后，在右侧查看：

- **Overview标签**：复制URL（API地址）
- **Headers标签**：找到 `Authorization` 或 `token`，复制值
- **Query String/Contents标签**：查看请求参数
- **Response标签**：查看响应数据格式

## ⚠️ 常见问题

**Q: 手机连不上代理？**
- 确保手机和电脑在同一WiFi
- 检查电脑IP是否正确

**Q: 抓不到HTTPS请求？**
- 确保已安装并信任证书
- iPhone需要在"证书信任设置"中开启

**Q: 找不到电费查询请求？**
- 在小程序中执行查询操作
- 搜索关键词：`power`、`电费`、`balance`
