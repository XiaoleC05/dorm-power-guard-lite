# 📱 微信小程序抓包教程（详细版）

## 🤔 什么是抓包？

**抓包**就是拦截和查看手机/电脑与服务器之间的网络通信数据。

### 简单理解

想象一下：
- 您用微信小程序查询电费时，小程序会向服务器发送请求："给我宿舍101的电费数据"
- 服务器返回数据："余额50元"
- **抓包工具**就像"中间人"，可以查看这些请求和响应的具体内容

### 为什么需要抓包？

微信小程序通常不提供公开的API文档，我们需要通过抓包来：
1. **找到API地址**：知道小程序向哪个网址发送请求
2. **获取认证信息**：提取Token、Cookie等认证凭证
3. **了解数据格式**：知道请求参数和响应数据的格式

## 📖 Charles抓包详细教程

### 第一步：安装Charles

1. **下载Charles**
   - 访问：https://www.charlesproxy.com/download/
   - 选择Windows版本下载
   - 安装（一路下一步即可）

2. **启动Charles**
   - 首次启动会提示注册，选择"14天免费试用"即可
   - 以后每次启动会询问是否继续试用，选择"Continue"即可

### 第二步：配置Charles抓HTTPS

1. **启用SSL抓包**
   - 菜单：`Proxy` → `SSL Proxying Settings...`
   - 勾选 `Enable SSL Proxying`
   - 在 `Include` 列表中添加：
     - Host: `*`（星号表示所有域名）
     - Port: `443`
   - 点击 `OK`

2. **查看Charles的代理端口**
   - 菜单：`Proxy` → `Proxy Settings...`
   - 记住端口号（默认是 `8888`）
   - 确保 `Enable transparent HTTP proxying` 已勾选

### 第三步：手机连接电脑代理

#### 方法A：手机和电脑在同一WiFi网络（推荐）

1. **查看电脑IP地址**
   - Windows：打开命令提示符（CMD）
   - 输入：`ipconfig`
   - 找到 `IPv4 地址`，例如：`192.168.1.100`
   - **记住这个IP地址**

2. **手机连接代理**
   - **Android手机：**
     - 设置 → WiFi → 长按当前连接的WiFi
     - 选择"修改网络"或"网络详情"
     - 高级选项 → 代理 → 手动
     - 主机名：输入电脑IP（如 `192.168.1.100`）
     - 端口：`8888`
     - 保存
   
   - **iPhone：**
     - 设置 → WiFi → 点击当前WiFi右侧的 `ⓘ`
     - 滚动到底部 → HTTP代理 → 配置代理
     - 选择"手动"
     - 服务器：输入电脑IP（如 `192.168.1.100`）
     - 端口：`8888`
     - 存储

3. **安装Charles证书（重要！）**
   
   **Android：**
   - 手机浏览器访问：`chls.pro/ssl`
   - 下载证书文件（通常是 `charles-proxy-ssl-proxying-certificate.crt`）
   - 设置 → 安全 → 加密与凭据 → 从存储设备安装
   - 选择下载的证书文件
   - 输入证书名称（如"Charles"）
   - 确定
   
   **iPhone：**
   - 手机浏览器访问：`chls.pro/ssl`
   - 点击"安装" → 允许
   - 设置 → 通用 → 关于本机 → 证书信任设置
   - 找到"Charles Proxy..." → 开启信任

### 第四步：开始抓包

1. **确保手机已连接代理**
   - 在Charles中，如果看到有请求出现，说明连接成功
   - 如果没有请求，检查：
     - 手机和电脑是否在同一WiFi
     - 代理设置是否正确
     - 证书是否已安装

2. **打开微信小程序**
   - 打开微信
   - 搜索并打开您的电费查询小程序
   - 执行查询操作（查询电费余额）

3. **在Charles中查看请求**
   - Charles左侧会显示请求列表
   - 找到包含"电费"、"power"、"balance"等关键词的请求
   - 点击请求，右侧会显示详细信息

### 第五步：提取关键信息

点击找到的请求后，在右侧查看：

#### 1. Overview（概览）标签
- **URL**：这就是API地址，例如：`https://xxx.com/api/power/query`
- **Method**：请求方法（GET或POST）

#### 2. Headers（请求头）标签
- 找到认证相关的字段：
  - `Authorization: Bearer xxxxxx`
  - 或 `token: xxxxxx`
  - 或 `Cookie: xxxxxx`
- **复制这些值**（这就是Token）

#### 3. Query String（GET参数）或 Contents（POST数据）标签
- 查看请求参数，例如：
  - `dorm=101`
  - `roomNumber=101`
  - 或POST数据：`{"dorm_number": "101"}`

#### 4. Response（响应）标签
- 查看返回的JSON数据格式
- 找到余额、用电量等字段

## 📖 Fiddler抓包教程（备选）

如果Charles使用有问题，可以使用Fiddler：

### 第一步：安装Fiddler

1. 下载：https://www.telerik.com/fiddler
2. 安装（免费版即可）

### 第二步：配置Fiddler

1. **启用HTTPS抓包**
   - Tools → Options → HTTPS
   - 勾选 `Capture HTTPS CONNECTs`
   - 勾选 `Decrypt HTTPS traffic`
   - 点击 `OK`，会提示安装证书，选择"是"

2. **查看端口**
   - Tools → Options → Connections
   - 记住端口号（默认是 `8888`）

### 第三步：手机连接（同Charles）

- 手机设置代理：电脑IP:8888
- 手机浏览器访问：`电脑IP:8888`（如 `192.168.1.100:8888`）
- 下载并安装证书

### 第四步：抓包

- 打开小程序，执行查询
- 在Fiddler左侧查看请求
- 找到相关请求，查看详细信息

## ⚠️ 常见问题

### Q1: 手机连接代理后无法上网？

**A:** 
- 检查电脑和手机是否在同一WiFi
- 检查电脑防火墙是否阻止了8888端口
- 尝试关闭电脑防火墙测试

### Q2: 抓不到HTTPS请求？

**A:**
- 确保已安装并信任证书
- iPhone需要在"证书信任设置"中开启信任
- Android需要从存储设备安装证书

### Q3: 找不到电费查询的请求？

**A:**
- 在小程序中执行查询操作
- 在Charles/Fiddler中搜索关键词：`power`、`电费`、`balance`、`余额`
- 查看请求的URL，找到包含API的请求

### Q4: Token在哪里找？

**A:**
- 通常在请求的 `Headers`（请求头）中
- 查找字段：`Authorization`、`token`、`X-Auth-Token`、`Cookie`
- 复制完整的值

### Q5: 抓包工具显示乱码？

**A:**
- 确保已正确配置HTTPS抓包
- 检查证书是否正确安装
- 尝试重启抓包工具
