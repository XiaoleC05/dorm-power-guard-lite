## 🚀 使用说明

### 配置邮件告警

1. **打开告警规则页面**
   - 访问：http://localhost:3000/alert-rules

2. **新增或编辑规则**
   - 点击"新增规则"或"编辑"按钮

3. **配置告警阈值**
   - 设置空调告警阈值（度）
   - 设置照明告警阈值（度）

4. **启用邮件告警**
   - 打开"邮件告警"开关
   - 输入接收邮箱地址
   - 支持多个邮箱（用逗号分隔）

5. **保存规则**
   - 点击"保存"按钮
   - 系统会验证邮箱格式

### 邮箱地址格式

**单个邮箱：**
```
user@example.com
```

**多个邮箱：**
```
user1@example.com, user2@example.com, user3@example.com
```

### SMTP配置

**需要在 `.env` 文件中配置：**
```env
EMAIL_ENABLED=true
EMAIL_SMTP_HOST=smtp.example.com
EMAIL_SMTP_PORT=587
EMAIL_SMTP_USER=your_email@example.com
EMAIL_SMTP_PASSWORD=your_password
EMAIL_FROM=your_email@example.com
```

**支持的端口：**
- 587：使用STARTTLS
- 465：使用SSL
- 25：普通SMTP（不推荐）

## ⚠️ 注意事项

### 邮箱验证

**验证规则：**
- 启用邮件告警时，邮箱地址必填
- 邮箱格式必须正确
- 多个邮箱用逗号分隔
- 每个邮箱都会验证格式

### SMTP配置

**必需配置：**
- SMTP服务器地址
- SMTP端口（587或465）
- SMTP用户名
- SMTP密码
- 发件人邮箱

**测试建议：**
- 先测试SMTP连接
- 确认邮箱地址正确
- 检查防火墙设置

### 邮件发送

**优先级：**
1. 规则中配置的邮箱地址（优先）
2. 全局配置的邮箱地址（备用）

**发送格式：**
- HTML格式（主要）
- 纯文本格式（备用）
